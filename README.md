# 📝 Task Manager Pro (GUI)

A simple and user-friendly Task Manager application built using Python and Tkinter.
This application allows users to manage daily tasks with features like add, update, delete, search, and mark tasks as completed.

---

## 🚀 Features

* ✅ Add new tasks
* ✏️ Update existing tasks
* ❌ Delete tasks
* ✔ Mark tasks as completed
* 🔍 Search tasks
* 🕒 Timestamp for each task
* 💾 Save tasks permanently using JSON file
* 🧹 Clear all tasks with confirmation
* 🎨 Simple and clean GUI

---

## 🛠️ Technologies Used

* Python
* Tkinter (GUI)
* JSON (File storage)

---

## 📂 Project Structure

TaskManager/
│── main.py
│── tasks.json (auto-created)
│── README.md

## ▶️ How to Run

1. Install Python (if not installed)
2. Download or clone the repository
3. Open terminal / command prompt
4. Run the program:

```bash
python main.py
```
